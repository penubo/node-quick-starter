### Spring Boot

Create basic setting of spring boot application. This will be the same as default setting on https://start.spring.io/
```bash
npx @penubo/qst
```

To see how to use
```bash
npx @penubo/qst -h
```

Interactive mode
```bash
npx @penubo/qst -i
```
